Project 1

triPPP

A collaborative travel application that allows users to vote on activities and create trips, has a map interface to see details of different cities within an area, with built in journal to capture specific moments of your trip. Utilized Google Maps API, Javascript, React/Redux, Git, Node, HTML5, CSS, Sequelize, PostgresSQL. Assumed an independent full-stack role responsible for both database setup and journal creation. Used Redux slices for state management and employed props to efficiently pass unique IDs to the journal.
